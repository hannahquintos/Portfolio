Spotlight Dance Competition

A content management system for a dance competition with create, read, update, and delete functionality for performances, dancers, and categories.

This application uses an ASP.NET model, view, controller (MVC) architecture pattern and uses entity framework code-first migrations to represent the database.

Built with C#, ASP.NET, HTML, CSS, JavaScript.
